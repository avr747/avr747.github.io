Archives Space Collection Access Report



For this assignment I was tasked with creating a finding aid for an unprocessed or partially processed collection, and in doing so learned about the challenges of making information available online. There are plenty of technical aspects information professionals must think of when making sure collections are accessible, such as making sure previous finding aids transferred over to a new system are able to be converted to a new program. I provided the VoiceThread presenation I did for this project, which better summarizes my experiences in working with a collection and building a finding aid using ArchivesSpace.
